PROJECTS

Uberlytics

This website tracks Uber surge fluctuation throughout the day and stores the data in a Firebase database. We then return this as a time graph using Chart.JS. In this way, we can help other users predict surge variations throughout the day and plan accordingly.

Perspective

Puzzle game developed with the Unity Game Engine. Players are thrown into a 3D maze environment and must manipulate gravity to walk along sideways corridors to find the exit. Players can use the Oculus Rift headset for a completely immersive experience. Check it out here.

IRC Server

Coded a server-client chat program in C++. Clients have unique usernames and passwords which allow them to perform actions such as creating rooms, sending messages, and seeing a list of all usersnames. The program runs in terminal.

Ballshake

Used the Unity Game Engine to create a simple mobile game for iOS and Android. The point of the game is to see how many times you can make a ball touch the edges of the screen by shaking your phone. A very simple yet addicting way to kill time.

SafeWalk

Wrote this Android application for my Object-Oriented Programming class. The purpose of the app is to provide safety by pairing students who are walking alone at night. Users can also request police escorts.

Hack Team Purdue

Co-founder of this club dedicated to helping students prepare for hackathons. The club helps form teams for club members attending the same hackathon or structures competitive teams to go and win prizes.
